# Operational Checklist

## Before run
- Confirm input data path exists and is readable.
- Validate configuration files in `configs/`.
- Verify Python environment and dependencies are installed.

## During run
- Monitor logs for warnings/errors every execution stage.
- Track CPU/RAM usage for unexpected spikes.
- Save intermediate artifacts for reproducibility.

## After run
- Validate output schema and record counts.
- Archive logs and key artifacts under a dated run folder.
- Update project notes with run context and next actions.
